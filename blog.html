what is DOM ? And what is the purpose of DOM ?

DOM stands for ducument object model . DOM is used to increase interactivity between user and the website. and it allows programs and scripts to dynamically access and update the content, structure, and style of a document.

How to select HTML Elements using DOM ? what are the names of DOM method ?

Using DOM it is possible to get access to HTML elements.Those methods are given below--
document.getElementById('IdName') > get access to the element by Id name.basically used to get one specific element.
document.getElementsByClass('ClassName') > get access to the element by class name .
document.getElementsByTagName('TagName') > get access to the element by tag name .
document.querySelector('#Idname/ .ClassName / TagName') > get access to the element by tag name or class name or Id name .
document.querySelectorAll(' .ClassName / TagName') > get access to the element by tag name or class name. it returns a nodelist which can be used to get access to each elemnts with similar class.

what is event bubble ?

bubble event is a part of DOM . It occurs when an event is given to an element and it also propagate the event to the parent element . to prevent this propagation , developers use stopPropagation() event Method .

what is callback function ? why is the used ?

A callback function is a function which is passed as a parameter to another function , and then it is called in another function to perform perticular activity. And the whole task can be perfomed sequencely.
And it allow a function to do the task , when the other function is excecuted. hence it prevents code repeatation, common errors and saves time of devs.
